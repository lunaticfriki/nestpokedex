# Start development server

1. Clone repository
2. Execute command

```pnpm
  pnpm install
```

3. Install Nest CLI

```pnpm
  pnpm add -g @nestjs/cli
```

4. Start database

```docker
docker-compose up -d
```

## Stack

- MongoDB
- NestJS
